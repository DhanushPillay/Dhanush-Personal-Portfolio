# Dhanush Pillay - Personal Portfolio

A highly interactive, deeply animated personal portfolio built for modern web standards. Showcases projects, skills, and experience with premium UI components featuring glassmorphism, 3D scroll effects, and Framer Motion animations.

## 🚀 Features

- **Premium UI Aesthetics**: Inspired by 21st.dev with glassmorphism and deep dark mode styling.
- **Interactive Animations**: Powered by `Framer Motion` and `GSAP` for scrolling timelines, pinning scroll triggers, and liquid buttons.
- **Bento Grid Layouts**: Modern, responsive grid systems to beautifully display stats and about information.
- **Infinite Marquees**: Seamless scrolling showcases for technical skills.
- **Dockerized**: Fully containerized for both production and development environments.
- **Vercel Ready**: Pre-configured with `vercel.json` and optimized build settings for one-click deployment.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion, GSAP, Lenis (Smooth Scrolling)
- **Icons**: Lucide React
- **3D**: Spline

## 📦 Running Locally

You can run this project using Node.js or Docker.

### Option 1: Using Node.js
Ensure you have Node.js installed, then run:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

### Option 2: Using Docker
Run the production build in a Docker container:

```bash
docker compose up -d --build
```
The portfolio will be available at `http://localhost:8080`.

## ⚙️ Deployment

This project is optimized for deployment on Vercel. 
1. Push this repository to GitHub.
2. Import the repository into your Vercel dashboard.
3. Vercel will automatically detect the Vite preset and handle the deployment.
